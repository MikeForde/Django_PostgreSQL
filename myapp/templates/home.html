Welcome to TT-MICP

This WebApp is designed to act as transitional tool as we move from DMICP to our future system.

It shows the underlying database structure of the data when stored and the input mechanisms.

It demonstrates key aspects of data/clinical code entry mechanisms to help apraise their current design and how they might be realised in the future.

It showcases some POC data input ideas as food-for-thought for future implementations.

  • View Consultation Data Examples
